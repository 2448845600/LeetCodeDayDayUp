## 项目结构

本项目主要包括工具箱、题型整理和题解三部分：

- 工具箱：主要讲解python3自带的库，如二分插入、排序等
- 题型整理：分门别类总结经典算法，如搜索、动态规划等
- 题解：顾名思义，Leetcode逐题解答
